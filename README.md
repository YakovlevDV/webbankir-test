# Описание:

Данный проект - тестовое задание на позицию Automation QA Engineer. В проекте выполнены задания 1, 2, 3.

# Библиотеки:

В проекте используется ряд библиотек:
- Java 21
- Maven 3+
- JUnit5
- Selenide
- WireMock
- Jackson

# Запуск тестов:

mvn clean test

В задании не было требования поднимать сервер или браузер - достаточно, чтобы тест был логически корректным, поэтому 
реальные зависимости замоканы через стабы WireMock

Можно так же передать настройки при запуске:
- -Dbrowser=chrome - целевой браузер для тестов, по умолчанию chrome
- -Dheadless=true - тип запуска браузера, по умолчанию в обычном режиме
